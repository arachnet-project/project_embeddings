🧬 Clinical Terminology Embeddings

 AI-driven clinical intelligence built on SNOMED CT, designed for real-world healthcare, scalability, and accessibility

Arachnet Project z.s. is a Czech-based non-profit initiative developing next-generation clinical intelligence tools. The project combines artificial intelligence, structured medical terminology, and cloud infrastructure to improve diagnostic workflows, reduce inefficiencies, and expand access to healthcare innovation — including for visually impaired professionals.

🌍 Vision

Arachnet aims to become a foundational layer for AI-assisted healthcare by enabling structured, interoperable, and accessible clinical reasoning systems. The long-term goal is to transform how medical knowledge is represented, processed, and applied in practice.

The project is uniquely shaped by accessibility-first design principles, developed in part by a blind engineer, ensuring inclusivity is built into the core architecture rather than added later.

🧠 What We Are Building

Arachnet Clinical Terminology Embeddings

A domain-aware clinical embedding platform built on SNOMED CT, designed to transform complex medical terminology into machine-understandable representations.

Unlike generic AI models, Arachnet uses a custom, clinically structured approach combining:

• textual representations of medical concepts
• ontology-based relationships from SNOMED CT
• domain-specific semantic weighting policies

This enables:

• advanced clinical reasoning support
• semantic interoperability across systems
• efficient querying of complex medical data
• integration with AI and machine learning pipelines

🧑‍⚕️ BlindPhysio

An AI-assisted diagnostic tool for physiotherapists, focused on musculoskeletal conditions and designed with accessibility as a primary requirement.

The system supports symptom-based reasoning and enables visually impaired practitioners to perform structured diagnostic workflows using standardized clinical terminology.

⚙️ Technology and Infrastructure

The platform is built on a modern, scalable stack designed for clinical-grade reliability:

• Oracle Cloud Infrastructure in Europe
• Oracle Database 23ai with native vector capabilities
• Python-based data and embedding pipelines
• SNOMED CT international terminology standard
• Linux environments across cloud and local development
• Secure Git-based development workflow

The architecture separates ingestion, validation, and inference layers, ensuring robustness, auditability, and regulatory readiness.

📈 Market Opportunity

Healthcare systems globally face major challenges:

• fragmented and inconsistent medical terminology
• lack of interoperability between systems
• increasing diagnostic complexity
• limited accessibility of digital tools

Arachnet addresses these gaps by combining standardized terminology with AI-driven semantic processing, enabling faster, more accurate, and more inclusive clinical decision-making.

🚀 Development Roadmap

The platform is being developed in structured phases:

• Foundation and infrastructure
• SNOMED CT ingestion and normalization
• semantic policy and domain modeling
• embedding engine and vector representations
• query and inference layer
• compliance, audit, and governance

This phased approach reduces risk while enabling early validation and iterative improvement.
